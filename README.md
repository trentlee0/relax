# relax

休息一下，专注、休息提醒 uTools 插件 & Web App

## 特性

1. 专注、休息提醒
2. 白噪音
3. 待办清单
4. 自定义背景
5. 专注休息时间线
6. 专注统计

## 安装

1. 在 uTools 插件应用市场搜索‘休息一下‘安装

2. 在 [Releases](https://github.com/trentlee0/relax/releases) 页面下载

## 构建

```shell
# 安装依赖
npm run install

# 运行项目
npm run serve

# 构建项目
npm run build
```
